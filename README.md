[![Build Status](https://travis-ci.org/dotink/inKWell-2.0.png?branch=master)](https://travis-ci.org/dotink/inKWell-2.0)


## Installation

```
git clone --recursive https://github.com/dotink/inKWell-2.0.git
cd inkwell-2.0
composer install
```
